Introduction Uploading large files like Excel sheets or images is a frequent requirement in many business applications. Previously, in RAP-based applications, the only way to handle such uploads was by extending the application and using UI5 tooling to support file uploads manually. However, with the release of SAP BTP ABAP environment version 2208, the RAP framework has introduced support for OData streams .
